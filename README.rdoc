= gmail_sender

A simple gem to send email through gmail

  require 'rubygems'
  require 'gmail_sender'

  g = GmailSender.new("gmail_account_user_name", "gmail_account_password")
  g.send("someone@domain.com", "The subject", "The mail body")

== Requirements

Ruby >= 1.8.7
